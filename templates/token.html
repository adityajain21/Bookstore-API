ADD API End Point

If you have the knowledge of Name and Price of Book you want to add, the end point is localhost:5000/add?name=XXX&price=XXX&flag=1




DELETE API End Point

If you have the knowledge of Name of Book you want to delete, the end point is localhost:5000/delete?name=XXX

If you have the knowledge of Lower and Upper range of price of Book you want to delete, the end point is localhost:5000/delete?lt=XXX>=XXX




SEARCH API End Point

If you have the knowledge of Name of Book you want to search, the end point is localhost:5000/find?name=XXX

If you have the knowledge of Lower and Upper range of prices of Book you want to search, the end point is localhost:5000/find?lt=XXX>=XXX
